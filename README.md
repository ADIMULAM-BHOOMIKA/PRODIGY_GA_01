# PRODIGY_GA_01
# Text Generation with GPT-2

## Objective
To generate human-like text using the GPT-2 language model and a custom dataset.

## Tools Used
- Python
- Google Colab
- Hugging Face Transformers
- GPT-2

## Steps Performed
1. Installed required libraries.
2. Loaded the GPT-2 pre-trained model.
3. Generated text from user prompts.
4. Created a custom dataset.
5. Generated text using the custom dataset.

## Outcome
Successfully generated coherent text using GPT-2 and demonstrated text generation based on custom input data.
